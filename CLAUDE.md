# CLAUDE.md

## デプロイ・起動手順を変更するとき

起動/停止フローに変更が生じた場合、以下の**すべて**を必ず同時に更新すること。

| ファイル | 更新内容 |
|---------|---------|
| `deploy.sh` | 起動スクリプト本体 |
| `stop.sh` | 停止スクリプト本体 |
| `flake.nix` | コンテナイメージのコメント・`shellHook` の起動案内 |
| `README.md` | `Run locally` / `Run as container` セクションの手順 |

一つでも漏れると手順が食い違うため、必ずセットで修正する。
